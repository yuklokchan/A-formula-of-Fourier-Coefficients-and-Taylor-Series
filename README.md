**A Formula Linking Fourier Coefficients and Taylor Series**

**Author**: Chan Yuk Lok (Form 2, Hong Kong)  
**Date**: May 2026

## About
This is my independent research project as a Form 2 student. I explored a hybrid formula that connects **Fourier Series** and **Taylor Series** in the interval `[-L, L]`. 

The project includes:
•Full 9-page report (PDF)
• LaTeX source code
• Convergence graphs
• Worked examples and a physics application

This project shows my passion for linking different areas of calculus. Even though the formula is mostly theoretical now, I hope it can inspire future applications in signal processing or physics.
